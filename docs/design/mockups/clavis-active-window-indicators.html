ACTIVE-WINDOW INDICATORS

Six ways to mark the active window once the title-bar dot is repurposed as a pure activity indicator (breathing while the agent works, dim when idle - shown in every mockup, identical in active and inactive windows so the two signals never mix). Each idea is shown as an active / inactive pair. All follow the design rules: square corners, content before chrome, one accent.

1 - Title-bar underline

A thin clavis rule under the active window's title bar, fading out to the right. Reuses the selected-tab underline vocabulary, so "active" reads the same at window and tab level.
