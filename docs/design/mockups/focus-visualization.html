CLAVIS · Focus Visualization Concepts

Six indicator concepts for window/panel and per-control focus, in the Clavis palette - elegant, minimal, square corners, no dashed frame. Below the gallery is a live, keyboard-driven demo of cross-window tabbing, focus skipping rules, slide-in focusability, and the arrow-driven permission prompt.

+Clear window/panel focus indicator
+Every control visualizes focus
+Chat input has its own indicator
+Cross-window tab when a window is exhausted
+Only interactive controls get focus
+Focus is never in a dead/keyless state
+Panels with no controls are never focused
+Parked slide-ins not focusable; open ones are
+Permission prompt never tab-focused, takes ← →
+No dashed-line frame anywhere
