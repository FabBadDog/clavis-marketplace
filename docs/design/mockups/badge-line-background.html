Badge - line background, varied text colours

The shared BadgeTemplate today fills the chip with FaintBrush #0E0E14 (near-invisible on the dark surface). This explores filling it with the line colour #5A5A66 (LineBrush/FrameBrush) instead, so the chip reads as a solid pill, then cycles the label colour through every brush in the default theme. Contrast ratios are WCAG 2.1 against #5A5A66; the badge label is small + semibold, so treat 4.5 as the "comfortable" bar and 3.0 as the floor for short uppercase words.
