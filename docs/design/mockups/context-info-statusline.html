The wrong hard-coded context % is replaced by real usage; the top bar drops the grip handle, the "CLAVIS" text and the idle/ready/thinking status, and becomes contextual per hosting panel. Rich, but never overloaded — graphical + minimal representations throughout. Same sample state everywhere so forms compare directly.

01 Placeholder model & syntax how every value is named & embedded

Templates are plain text/markdown with {namespace.key} tokens. Three token shapes: value {git.branch}, parameterised value {time.now:HH:mm}, and component {bar:agent.contextPercent} / {limitPlane} / {spark:sys.cpu} which render a widget inline. Unknown tokens are left verbatim (today's command-palette/Placeholders.cs already does this — we generalise it to dotted namespaces + components).
