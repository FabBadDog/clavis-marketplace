CLAVIS — Debug Panel

Three design variants for a real-time debug stream showing StreamEvents, ParsingErrors, and SessionInputs with timestamps. Each row is categorized by origin and color-coded. The panel should live as a toggleable overlay or docked panel.

Single chronological stream. Color-coded rail dots mark category. Filter chips toggle visibility. Minimal — every pixel serves the debugging task.
