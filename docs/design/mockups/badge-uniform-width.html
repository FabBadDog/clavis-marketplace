CLAVIS / uniform-width badges

When a column of badges of the same family appears together (event levels, command kinds…), they all take the width of the widest label and the shorter ones center - so INFO and WARN gain equal padding left and right. Padding is your chosen 6,3. This is opt-in per list, not global: a lone badge keeps its natural width.

The four levels, equalized

Left: natural widths (ragged right edge). Right: uniform - every chip matches the widest (DEBUG/TRACE/ERROR), text centered. The dashed line marks the shared right edge.

How it's built: set Grid.IsSharedSizeScope="True" on the list (the events / palette ItemsControl) and give the badge's column SharedSizeGroup="BadgeWidth"; the badge Border stretches to fill it with its label centered. WPF sizes the column to the widest badge automatically - no fixed number, and it only applies inside that one scope.
